# Diwali-Sales-Analysis
The "Diwali_Sales_Analysis" project uses Python and its libraries to analyze sales data around the Diwali festival. It involves data collection, cleaning, and preprocessing using libraries like Pandas and NumPy. Exploratory data analysis and visualization are performed with Matplotlib and Seaborn. Sales forecasting and customer segmentation leverage machine learning libraries such as Scikit-learn.
